# Kolomyia Elders

## Local development

1. `npm run preview` to view.
1. Optionally in a separate terminal: `npm run build` to watch.

## Deploy

1. Run `npm run preview`
1. Add `.` to the paths in `.\dist\index.html`. TODO: remove the need for this.
1. `npm run deploy`

## Upgrade packages

`npx npm-check-updates --upgrade`
